# synthetic-unknown-game

Hand-authored, non-copyrighted fixture used by `restrike detect`'s tests.
Deliberately does not resemble real Gold Box file names or formats
(no `.DAX`, `.GEO`, `ECL*`, `SAVGAM*`, etc. — see the repo `.gitignore` and
the CI no-game-data guard) so it can never be mistaken for real game data.
